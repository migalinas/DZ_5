# Как использовать CMake для запуска проекта.
## 1.Загружаем проект с GitHub(``` git clone https://github.com/serpenttttt/lab_5.git ```)
## 2.Используем следующие команды:
### 1)
``` cd DZ_5 ```
### 2)
``` git clone https://github.com/serpenttttt/lab_5.git ```
### 3)
```mkdir build```
### 4)
```cd build```
### 5)  
```cmake ..```
### 6)  
```cmake --build .```
